### Критерии для старта

-  Готово десктопное приложение без необходимости ходить в VSCode.

-  Есть подключения к разным хранилищам: GitHub, GoogleDrive, OneDrive, Яндекс.Диск.

-  Есть мобильное приложение для Android и IOS.
